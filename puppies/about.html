About Best in Show Puppies

Why I Created This Page

I created this page to show how a simple classroom assignment can become a full and meaningful website section. Instead of only displaying basic Bootstrap columns, I wanted to turn the About page into something that feels more complete, personal, and connected to the puppy theme.

Puppies bring joy, comfort, and personality to a page, so I wanted the design to feel welcoming and fun. This project helped me practice combining images, text, layout, and responsive design into one organized webpage.

Best in Show Puppies is a cheerful puppy-themed website created for my Web Fundamentals final project. This page was designed to feel warm, friendly, and easy to explore while also demonstrating responsive web design, Bootstrap rows and columns, image placement, and organized page structure.

Puppy Care

Puppies need patience, love, attention, and a safe environment to grow. This section represents the importance of care and responsibility when raising animals.

Companionship

Dogs are loyal companions that bring happiness and comfort into people’s lives. Their playful personalities make them special and memorable.

Web Design

This page demonstrates Bootstrap columns, responsive image placement, and organized page content while keeping the design friendly and easy to view.

What I Learned

Building this page helped me understand how Bootstrap grids can turn a simple layout into a responsive and organized webpage. I learned how rows, columns, images, spacing, and written content work together to create a better user experience.

My goal was to take a basic assignment page and redesign it into a complete About page that feels fun, polished, and connected to the overall website theme.
