Web Designs

Beyond the lines of code and technical specifications, I believe in building digital spaces that feel welcoming, creative, and meaningful. As I continue learning web development, I enjoy combining structure, styling, and responsive design to create clean and engaging user experiences.

The projects below showcase my growth in HTML, CSS, Bootstrap, responsive layouts, image galleries, and modern web design principles. Each project represents another step forward in my journey toward becoming a software developer and web designer.

Groove Guitars

Groove Guitars is a music-themed website created to practice layout design, tables, images, navigation, and styling. This project demonstrates how color, branding, and content can work together to create a strong website theme.

Animal Photography

This project showcases an interactive animal photography gallery using HTML, CSS, Bootstrap, and JavaScript. The page focuses on visual storytelling, peaceful nature photography, and responsive image layouts.

Best in Show Puppies

Best in Show Puppies was created to demonstrate Bootstrap rows and columns, responsive image placement, and a friendly visual theme. The project uses puppy images, structured content, and simple navigation to create a welcoming web page.
